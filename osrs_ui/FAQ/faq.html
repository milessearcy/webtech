FAQ
How do I search for items?

Simply just type the item you wish to search.

Where do the prices come from?

An API that can be found on the offical OSRS wiki.

How often is the data updated?

The data table is updated every 5 minutes, meanwhile, the individual item pages are updated when the page loads

Are all items included?

No. Sadly this only shows the items sold in the last 5 minutes. In order to get a accurate amiunt of data i would need to run a data collection script for over a year and even then it wouldn't be 100%
